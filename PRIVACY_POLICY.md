# 개인정보 처리방침 (Privacy Policy)

**태해린**(이하 '회사')은(는) **가치록 (Gachi-Rok)** 앱(이하 '앱')을 이용하는 이용자의 개인정보를 보호하고, 관련 법령에 따라 이용자의 권익을 보호하기 위해 다음과 같은 개인정보 처리방침을 수립·공개합니다.

## 1. 개인정보의 처리 항목 및 수집 방법
앱은 이용자의 민감한 개인정보를 서버로 전송하거나 수집하지 않는 것을 원칙으로 합니다.

* **메모 데이터**: 이용자가 작성한 메모 내용은 앱 내부의 로컬 데이터베이스(Room DB)에만 저장됩니다.
* **백업 파일**: 이용자가 직접 실행하는 JSON 백업 및 복구 기능을 위해 기기 내 저장공간을 활용하며, 이 파일은 이용자의 기기 외부에 자동으로 전송되지 않습니다.
* **생체 인증 (지문 인식)**: 앱 잠금 해제 기능을 위해 안드로이드 표준 생체 인식 API를 사용합니다. 이 과정에서 지문 데이터는 앱에 저장되거나 전송되지 않으며, 안드로이드 OS의 안전한 영역(Secure Enclave)에서 인증 결과만 전달받습니다.

## 2. 개인정보의 처리 목적
앱은 수집된 정보를 다음의 목적을 위해서만 사용합니다.
* 서비스 제공 및 관리: 메모 작성, 저장, 수정 및 삭제 기능 제공.
* 데이터 관리: 이용자의 요청에 의한 JSON 파일 기반 데이터 백업 및 복구.
* 보안: 생체 인증을 통한 앱 접근 제한 및 보안 강화.

## 3. 개인정보의 제3자 제공
회사는 이용자의 개인정보를 원칙적으로 외부에 제공하지 않습니다. 모든 데이터는 이용자의 기기 내에 로컬로 유지됩니다.

## 4. 개인정보의 안전성 확보 조치
앱은 이용자의 데이터를 보호하기 위해 다음의 노력을 하고 있습니다.
* 로컬 저장: 데이터가 외부 서버에 저장되지 않으므로 서버 해킹으로 인한 유출 위험이 없습니다.
* 표준 API 사용: 안드로이드 표준 생체 인식 프레임워크를 사용하여 보안성을 확보합니다.

## 5. 이용자의 권리와 행사 방법
이용자는 언제든지 앱을 삭제하거나 앱 내 데이터를 초기화함으로써 본인의 정보를 스스로 관리할 수 있습니다. 백업 파일(JSON) 또한 이용자가 직접 관리하고 삭제할 수 있습니다.
