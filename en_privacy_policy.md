# Privacy Policy

Taehaerin ("Company," "we," "us," or "our") is committed to protecting the privacy of users of the 'Gachi-Rok' app (the "App"). This Privacy Policy outlines how we handle information in compliance with relevant laws.

### 1. Data Collection and Method
The App does not collect or transmit sensitive personal information to external servers.
* **Memo Data:** All content is stored only in the local database (Room DB) on the User's device.
* **Backup Files:** JSON files for backup/recovery are stored on the device's local storage and are not automatically transmitted externally.
* **Biometric Authentication:** The App uses standard Android Biometric APIs for app locking. Fingerprint data is never stored in the App; we only receive the authentication result from the OS's Secure Enclave.
* **Payment Information:** We do not collect or store credit card or bank account details. All transactions are handled securely by Google Play.

### 2. Purpose of Processing
The App uses information solely for the following purposes:
* **Service Management:** Providing features for creating, saving, and editing memos.
* **Data Portability:** User-initiated data backup and restoration using JSON files.
* **Security:** Restricting app access through biometric authentication.

### 3. Data Retention and Destruction
* Since all data is stored locally, the Company does not retain or manage User data.
* **All data is immediately destroyed when the User deletes the App or clears app data.** Users are responsible for deleting any manually created JSON backup files.

### 4. Third-Party Disclosure
The Company does not provide User information to third parties, except for payment processing handled by Google Play (Google LLC) during a Premium subscription purchase.

### 5. User Rights
Users have full control over their data and can manage it at any time by deleting the App, clearing data, or managing their own JSON backup files.
